# 💫 About Me:
🚀 About Me<br>Hey Kirthan here🤗, I am an Engineering Student with a strong interest in robotics, embedded systems, and software development. My work focuses on building and integrating systems that combine hardware and software, with an emphasis on practical implementation and problem-solving.<br>I have experience working with microcontroller-based systems, including Arduino UNO & MEGA, STM32 and ESP8266, and I am currently exploring ROS 2 and distributed communication frameworks for robotics applications. My projects involve areas such as robot kinematics, URDF modeling for multi-DOF manipulators, and wireless communication between devices.<br>In addition to hardware-oriented work, I have a solid foundation in data structures and algorithms in C, including stacks, queues, and linked lists. I am developed intermediate skills in Fusion 360 for design and prototyping.<br><br>Currently, I’m working on:<br>🤖 Robot control and kinematics (including URDF modeling and multi-DOF arms)<br>🔌 Embedded systems using Arduino, ESP8266, and microcontroller-based boards<br>🌐 ROS 2 and communication systems (WiFi-based control, micro-ROS exploration)<br>🧠 Data structures and algorithms in C (queues, stacks, linked lists, etc.)<br>🛠️ Real Time CAD to URDF convertion and simulation tools in Fusion 360<br>I like debugging real-world problems—whether it’s fixing mesh assemblies, getting drivers to work on Linux, or making hardware talk to software reliably.<br><br>💡 What I’m Interested In<br>Robotics<br>Automated Systems<br>Embedded + IoT systems<br>Designing CAD Models<br>PCB Designing<br><br>🧰 Tech Stack<br>Languages: C++, JAVA, Python, Embedded C<br>Tools: Arduino IDE, ROS 2, Fusion 360, Ubuntu/Linux, Google Colab, Jupyter, ABB Robot Studio, ROS2<br>Concepts: Data Structures, Control System fundamentals, Robot Kinematics & Dynamics, ML & DL Model <br><br>⚡ How I Work<br>I learn by building, breaking things, and fixing them.
